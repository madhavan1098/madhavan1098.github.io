Social Proof Popup Builder

Using social proof popup builder you can create social proof popups for websites which help in increasing trust and confidence in the minds of customers for brands. Mainly useful for ecommerce websites as well as websites selling digital goods or other digital services.When a social proof popup is integrated in website say for eg: an ecommerce website, when any user purchases a product from website a popup will be displayed on website showing that a user has purchased a product from website which will display name of user location and product details. When such purchases are seen by other website users it generates an urgency to buy as well as trust & confidence in the brand.

Technologies

PHP, Laravel,SQL, AWS, HTML - CSS- JavaScript

Responsibilities

Tasked with preparing product roadmap and product requirement document, Responsible for all UX/UI design ( User Flow Diagrams, Wireframes and Mockups) Running agile sprints, writing user stories and working with the development team to make sure the project is delivered on time.
