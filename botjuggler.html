Botjuggler

Botjuggler is a Do It Yourself chatbot builder which enables anyone to build chatbots for websites without knowing any coding skills. All the user needs to do is sign up for an account, select a chatbot template of choice, customize the look and feel of chatbot according to his requirement, use the easy to use builder to set up chatbot conversations once done click publish button to generate chatbot script to place in the website.

Technology

PHP, Laravel, AWS, HTML - CSS- JavaScript
